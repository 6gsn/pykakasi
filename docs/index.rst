Welcome to pykakasi's documentation
====================================

pykakasi is a library and utility implemented `KAKASI` utility in Python.
KAKASI was originaly built to convert Japanese text to roman form.

pykakasi is a free software, and available on `GitHub`_.

.. _GitHub: https://github.com/miurahr/pykakasi
